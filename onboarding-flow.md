# Consulting 101 — Onboarding Flow

## Step 1: Free consult (30 min)
- Goal: qualify fit + identify 1–2 workflows
- Output: quick notes + next‑step decision

## Step 2: Paid audit (1–2 weeks)
- Map the workflow end‑to‑end
- Identify automations + dependencies
- Estimate time saved + ROI

## Step 3: Proposal
- Fixed scope + timeline + price
- Deliverables list + success criteria

## Step 4: Build & install (2–4 weeks)
- Implement OpenClaw workflows
- Integrate tools (email/CRM/Docs/Telegram)
- Test with real data

## Step 5: Handover
- Training session
- Documentation
- 2‑week bug‑fix window

## Step 6: Retainer (optional)
- Monthly improvements + new workflows
- Monitoring + updates
